== Installation ==

1. Upload the entire `ai-contact-us` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use shortcode [contact_us] on the page you want to display.

You will find 'Contact Form' menu in your WordPress admin panel
